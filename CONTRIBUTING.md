# Contributing to the *MV Hondius* Outbreak Timeline

Thank you for contributing.

This project aims to maintain a structured, source-based chronology of the *MV Hondius* outbreak and related public health response activities.

---

# Contribution Principles

Contributions should strive to be:

* factual
* source-based
* chronological
* transparent
* non-speculative

Where possible, events should be traceable to publicly available primary or secondary sources.

---

# Before Contributing

Please:

* review existing events
* reuse existing tags where appropriate
* avoid duplicate entries
* maintain consistent naming conventions

---

# Event Writing Style

Event descriptions should be concise and source-faithful.

Preferred:

```text
Case #3 was medically evacuated from Ascension Island to Johannesburg.
```

Avoid:

```text
The outbreak spread internationally after failures in onboard containment.
```

unless explicitly supported by cited sources.

---

# Categories

Categories are intentionally broad.

Current categories:

* `cases`
* `contacts`
* `itinerary`
* `outbreak`

Please avoid creating new categories unless:

* existing categories are clearly insufficient
* the distinction is meaningful
* the category is likely to recur

If uncertain, open an issue for discussion.

---

# Tags

Tags should:

* use lowercase kebab-case
* remain concise
* reuse existing terminology where possible

Examples:

```text
south-africa
case-003
onset
death
```

Avoid:

* duplicates
* abbreviations without context
* inconsistent formatting

---

# Sources

All substantive events should include at least one source.

Preferred sources include:

* WHO notices
* CDC notices
* government/public health communications
* peer-reviewed publications
* reputable journalism
* official organizational statements

---

# Pull Requests

Pull requests are welcome for:

* new events
* corrections
* source improvements
* metadata cleanup
* visual improvements
* accessibility improvements
* documentation improvements

Please include a brief description of major changes.

---

# Questions or Suggestions

If you are unsure about:

* categories
* tags
* event wording
* source suitability
* chronology
* or larger structural changes

please open an issue for discussion before making large changes.

Thank you for contributing.
